Spring Web MVC Request-Mapping

1. 메소드에 요청 매핑하기!

GET 방식의 요청 : /menu/regist

POST 방식의 요청 : /menu/regist

GET 방식의 요청 : /menu/modify

POST 방식의 요청 : /menu/modify

2. 클래스에 요청 매핑하기!

GET 방식의 요청 : /order/regist

POST 방식의 요청 : /order/regist

GET 방식의 요청 : /order/modify

POST 방식의 요청 : /order/modify

POST 방식의 요청 : /order/delete

GET 방식의 요청 : /order/detail/{orderNo}
